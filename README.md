# Requisitos

- Intalar Nodejs

# Execução
- Entrar na pasta:
```shell
cd plataforma/apps/sistema
```
- Instalar bibliotecas e executar sistema:
```shell
npm install
npm start
```
visitar localhost:3000

- Gerando ambiente de produção para integração com o Spring Framework:
```shell
npm run build
cd ../../
mvn spring-boot:run
```
visitar localhost:8080
